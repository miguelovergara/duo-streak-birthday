## âœ¨ Features

- **Milestone Calculation:** Automatically calculates upcoming milestones in days (100, 250, 500...), months, and years.
- **Real-time Updates:** See your timeline update instantly as you enter your current streak.
- **Motivational Messages:** Receive personalized encouragement based on your progress.
- **Duolingo-inspired UI:** A clean, playful interface that matches the Duolingo aesthetic, built with Tailwind CSS.
- **Date Localization:** Milestone dates are clearly formatted (currently supporting Spanish localization).

## ğŸ“¦ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash

   cd duolingo-streak-birthdays
   ```


   ```bash
   npm run build
   ```
