**Duo Streak Birthday** is a React-based web application designed for Duolingo enthusiasts. It calculates and displays your future streak milestones, helping you stay motivated and plan for your next big "streak birthday".

## üöÄ Live Demo

Check out the live application here: [duo-streak-birthday.vercel.app](https://duo-streak-birthday.vercel.app/)

## ‚ú® Features

- **Milestone Calculation:** Automatically calculates upcoming milestones in days (100, 250, 500...), months, and years.
- **Real-time Updates:** See your timeline update instantly as you enter your current streak.
- **Motivational Messages:** Receive personalized encouragement based on your progress.
- **Duolingo-inspired UI:** A clean, playful interface that matches the Duolingo aesthetic, built with Tailwind CSS.
- **Date Localization:** Milestone dates are clearly formatted (currently supporting Spanish localization).

## üì¶ Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/miguelovergara/duo-streak-birthday.git
   cd duolingo-streak-birthdays
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
